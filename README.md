# AgriVision Analytics

AgriVision Analytics is an interactive Tableau dashboard designed to analyze agricultural performance across yield, revenue, profit, weather impact, and supply chain efficiency.

---

## Project Overview

This project provides an end-to-end data visualization solution for agricultural analytics. It converts raw data into meaningful insights through interactive dashboards and a structured layout.

> Note: The dataset used in this project is for demonstration purposes and does not represent real-world data.

---

## Key Features

- 5 interactive dashboards:
  - Agriculture Overview
  - Weather & Environmental Impact
  - Market & Profit Analysis
  - Farm Operations
  - Supply Chain Performance

- KPI tracking:
  - Total Yield
  - Total Revenue
  - Total Profit

- Interactive filters:
  - Crop-based analysis
  - District-level insights
  - Time-based trends

- Consistent dark-themed dashboard design
